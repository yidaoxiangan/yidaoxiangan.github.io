西方哲学导论 Week3 Note

Posted on 2021-03-05 Edited on 2021-08-01

week3_Note

前苏格拉底时代

Anaximander

  • 万物的本原是“无定”
  • 万物的多样性，水的“湿性”不能解释一切
  • 万物的本原应是没有任何确定规定性 - 混沌
  • 不定形且与水有关的东西 为 无定

Anaximenes 546 a.d. - 545 a.d

  • 世界的本原是气（综合了水和“无定”的特征）
  • 与东方本体论相似

Heraclitus 赫拉克利特 504 a.d. - 501 a.d

  • 性格高傲，出身望族， 把王位让给兄弟
  • “世界是客观的”
  • “世界的本原是火” - “世界秩序不是任何人或任何人锁创造的，它过去，现在，未来永远是永恒的活火，在一定分寸上燃烧，在一定分寸上熄灭”
    • 唯物主义 - 客观存在
    • 万事万物是变化的
  • “万物皆变，无物常驻”
    • “一切都是通过斗争和必然性而产生的”
    • “太阳每天都是新的”
    • “人不能同时走进同一条河流”
    • “我们存在又不存在”
  • 简单，直观，朴素的表达

本原内在不变的统一原则是“逻各斯”

原意为话语，指说出的道理，是火在燃烧中遵循的理性规则和原则。即“一定分寸”

我们看见特殊，我们只能说出普遍

  • 我们使用名词描述 - 名词 在 逻辑学上 为 概念

毕达哥拉斯学派

毕达哥拉斯 建立了一个有伦理，宗教和政治目的的社团：大家庭式的集体生活。

基本观点：数是万物的本原

  • 第一，数先于事物存在，是构成万物的基础
    • " 万物的本原是一，从一产生二，而是从属于一的补丁的质料，一则是原因。.....”
  • 第二，一切事物的性质都可以被归结为数的规定性
    • 比例关系：（决定事物结构的和谐）
    • 对立关系：（有限和无限，奇数和偶数）
    • 类比关系：（4代表正义，8代表爱情.......）

埃利亚学派

创始人：Xenophanes

  • 世界的本原是绝对不动的“神”
